II. Based on System Knowledge

3. Adaptive Control

Used when system parameters are unknown or changing; controller adjusts in real time.

  • Model Reference Adaptive Control (MRAC)
  • Adaptive Backstepping
  • Gain Scheduling

4. Robust Control

Designs for the worst-case uncertainty; stability is ensured even with bounded disturbances.

  • H-infinity Control
  • Sliding Mode Control (again)
  • Œº-synthesis

III. Based on Domain (Time vs Frequency)

5. Time-Domain Control

Works in state-space form and is more suited for modern nonlinear systems.

  • State feedback
  • Lyapunov theory
  • Backstepping

6. Frequency-Domain Control

Used mostly for linear time-invariant (LTI) systems.

  • Bode Plot
  • Nyquist Plot
  • Root Locus

IV. Other Specialized Control Techniques

7. Optimal Control

Minimizes a performance cost function over time.

  • LQR (Linear Quadratic Regulator)
  • MPC (Model Predictive Control)
  • Dynamic Programming

8. Intelligent Control

Uses AI techniques to handle complex or poorly modeled systems.

  • Fuzzy Logic Control
  • Neural Networks
  • Reinforcement Learning

9. Geometric Control

Uses differential geometry and Lie groups for controlling systems like spacecraft, drones, and AUVs.

  • SE(3), SO(3) manifold control
  • Non-Euclidean feedback design

Summary

Control theory is the mathematical and engineering framework that helps regulate and guide the behavior of dynamic systems. It's used not only for robots and vehicles but also in finance, biology, energy systems, and more.
